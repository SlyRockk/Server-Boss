Server Boss is a server manager for Team Fortress 2 designed to make the process of setting up a tf2 server as easy and simple as possible, while also being flexible and customizable.

> [!CAUTION]
> This tool is currently in its alpha version, so you may encounter some bugs while using it.

## Key features 
- 2 clicks server setup
- FakeIP support(avoids the need for port forwarding)
- drag and drop support for custom maps and map images
- simple UI
- customizable console(support for custom text colors and size)
- bot commands presets

## OS Support
- Windows 10
- WIndows 11

## Setup
To create a TF2 server, simply open `Server Boss v0.1.exe`. The program will ask you to setup a server, click yes and wait until steamCMD downloads everything.

If you already have server setup then copy all files from the realease to the folder that contains `srcds_win64.exe`.

## How to compile
- Download this repository
- Download Autohotkey v1.1 (deprecated) from [www.autohotkey.com](https://www.autohotkey.com/)
- Install the program
- Right click on `Server Boss v0.1.ahk` and select Compile Script (GUI)...
- Change the Base File (.bin, .exe) to v1.1.37.02 U64 AutoHotkeyU64.exe
- Press compile
