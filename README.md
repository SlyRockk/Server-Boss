The easiest tool for setting up and managing a Team Fortress 2 servers

gif goes here

## Key features 
- 2 clicks server setup
- FakeIP support(no need for server port forwarding)
- drag and drop support for custom maps and map images
- simple UI
- customizable console(support for custom text colors and size)
- bot commands presets

## Usage
Open `Server Boss.exe` to download steamCMD and setup up a tf2 server(Yeah, that's the whole process)
If you already have a server then just copy all of the files in the same folder as `srcds_64.exe`
